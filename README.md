# 🤖 Question Generator Chatbot (LLMs)

This project is a **simple Question & Answer chatbot** built using the **Google Generative AI package**.
It generates questions based on a given topic using Large Language Models (LLMs).

---

## 🚀 Features

* Generate questions from any topic
* Uses LLMs for intelligent question generation
* Simple and beginner-friendly implementation
* Customizable prompts

---

## 🛠️ Tech Stack

* Python
* Google Generative AI (Gemini API)
* LLMs

---

## 📦 Installation

1. Clone the repository:

```
git clone https://github.com/manikanta33124/Chatbots-using-LLMS.git
cd Chatbots-using-LLMS
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Add your API key:

* Get API key from Google AI Studio
* Set it in your code or environment variables

---

## ▶️ Usage

Run the script:

```
python app.py
```

Enter a topic, and the chatbot will generate questions based on it.

---

## 💡 Example

**Input:**

```
Topic: LLMs
```

**Output:**

```
1. What are Large Language Models?
2. How do LLMs work?
3. What are the applications of LLMs?
...
```
